*AgroCahs – Sistema de Controle Financeiro para Agronegócio*

O AgroCahs é um sistema em Java desenvolvido para auxiliar no controle financeiro de atividades agrícolas. Ele permite cadastrar usuários, gerenciar lavouras e registrar despesas, facilitando a organização dos dados no agronegócio.

O projeto segue uma arquitetura em camadas inspirada no padrão MVC, separando lógica de negócio, controle e interface. Utiliza conceitos de programação orientada a objetos e persistência de dados por meio de serialização em arquivos locais.

Entre as funcionalidades estão o cadastro de usuários com validação de CPF, associação de lavouras, registro de despesas e listagem de informações. Os dados são armazenados em arquivos .dat, organizados por usuário.

O sistema foi desenvolvido com foco em aplicar boas práticas como encapsulamento, separação de responsabilidades e uso de estruturas como Map e List.
